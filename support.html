Head over to the forums here: (Dump Chat Forums)

Problems?

As much as software developers would like to have a perfect, bug free release, it's not always possible to find all the bugs before a product launch. Dump Chat includes an error reporter which detects crashes on the next time you launch it. When a crash is detected, Dump Chat will ask you if you would like to send the crash report to the author. No sensitive data is sent to our server. Only the program trace is sent back to us (meaning the memory locations when the program crashes). We never collect any information regarding the user. Pressing the send button after a crash will help us to make a better product for you for future updates.

If you have any problems with the software itself, such as a feature not doing what it should, then we definitely want to hear from you. You can send us an e-mail or post a message on the forums over here (Dump Chat Forums). We check both e-mail and the forums often, so choose what's more convienent for you.
